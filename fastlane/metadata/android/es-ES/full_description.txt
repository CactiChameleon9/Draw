Características de Simple Draw 

Este popular cuaderno de bocetos para dibujar y pintar rápidamente ofrece muchas funciones útiles:

✔️ Dibuja algo colorido, bocetos o garabatea usando una pintura y un tamaño de pluma diferentes
✔️ Cambiar el color de fondo, o usar una imagen de sus bocetos como fondo 
✔️ Utilice una goma de borrar si tomó la pintura errónea
✔️ Insertar pintura seleccionándola en una paleta, o con el código hexadecimal de colores
✔️ Este sencillo cuaderno de bocetos soporta muchos formatos diferentes PNG, JPG, o vectores SVG
✔️ Comparte tus pinturas, dibujos y bocetos con amigos a través de correo electrónico o redes sociales
✔️ ¡Simple Draw funciona sin conexión y en línea!
