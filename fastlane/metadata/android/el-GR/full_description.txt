Θέλετε να σχεδιάσετε κάτι δημιουργικό αλλά δεν έχετε χαρτί; Αυτή η εφαρμογή θα σας ταιριάξει τέλεια. Απλά επιλέξτε ένα χρώμα και σχεδιάστε, είναι τόσο εύκολο. Μπορείτε είτε να αποθηκεύσετε το σχέδιο σε ένα αρχείο, να το εκτυπώσετε ή να το μοιραστείτε απευθείας μέσω ηλεκτρονικού ταχυδρομείου ή κοινωνικών δικτύων, έτσι ώστε οι φίλοι σας να μπορούν να διασκεδάσουν μαζί του.

Αυτή η εξαιρετικά δημοφιλής εφαρμογή για γρήγορα σκίτσα παρέχει πολλές χρήσιμες λειτουργίες όπως η αναίρεση/επανάληψη, η αλλαγή του χρώματος φόντου ή η χρήση μιας εικόνας ως φόντου. Μπορείτε προαιρετικά να κάνετε την εφαρμογή να εμποδίζει τη συσκευή να αδρανοποιηθεί επίσης. Μπορείτε επίσης να αλλάξετε το μέγεθος του πινέλου, ή να χρησιμοποιήσετε μια γόμα, αν κάνατε κάτι λάθος. Υποστηρίζει την εισαγωγή χρωμάτων είτε με την επιλογή του απο μια τεράστια παλέτα, είτε μέσω χρωμάτων hex κώδικα.

Η εφαρμογή υποστηρίζει πολλές διαφορετικές μορφές, όπως PNG, JPG ή SVG.

Μπορείτε είτε να σχεδιάσετε κάτι νέο από το μηδέν, είτε να ανοίξετε κάποιο αρχείο και απλά να ζωγραφίσετε σε αυτό. Είναι απόλυτα χρησιμοποιήσιμο είτε από εσάς, είτε από οικογένεια με παιδιά για να παίξετε μερικά παιχνίδια. Μετά, όλοι έχουν ένα κομμάτι καλλιτέχνη στον εαυτό τους.

Δεν θα βρείτε τα φανταχτερά προηγμένα εργαλεία, βούρτσες, γραμματοσειρές, φίλτρα, θα τα χρησιμοποιήσετε για να διασκεδάσουν.

Διατίθεται με σχεδίαση υλικού και σκούρο θέμα από προεπιλογή, παρέχει εξαιρετική εμπειρία χρήστη για εύκολη χρήση. Η έλλειψη πρόσβασης στο διαδίκτυο σάς παρέχει περισσότερο απόρρητο, ασφάλεια και σταθερότητα από ό,τι άλλες εφαρμογές.
